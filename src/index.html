Bootstrap + Vite Multipage

Ini adalah kit dasar untuk membuat website semi dinamis dengan bundler Vite yang dapat mempermudah Anda dalam membangun situs web semi dinamis dan multi-halaman dengan mudah dan cepat. Dilengkapi dengan berbagai ikon dan plugin tambahan, memungkinkan Anda membuat dan membangun situs web lebih cepat dan lebih keren.

Kit Permulaan ini juga dilengkapi dengan perkakas kompiler scss yang mana akan mempermudah pengembangan tampilan lebih cepat ketimbang css biasa.
